# Eslint-config-atol

## Install

npm: 

```bash
npm i @atol/eslint-config-atol --save-dev
```

or yarn: 

```bash
yarn add @atol/eslint-config-atol --dev
```

# Usage

Update `.eslintrc.js`:

```js
module.exports = {
    extends: ['@atol/eslint-config-atol'],
    rules: {
      // ... Add your rules
    }
};
```
